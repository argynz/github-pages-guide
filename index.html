Claude: шаг 1 из 5

Шаг 1

Создайте аккаунт Claude

Для тренинга нужен один основной ИИ-сервис. Лучше авторизоваться через : так проще входить на разных устройствах и меньше риск забыть пароль.

  1. Откройте официальный сайт: claude.ai.
  2. Нажмите вход или регистрацию.
  3. Выберите вход через , если такая кнопка доступна.
  4. Введите email, к которому у вас есть доступ.
  5. Вернитесь на сайт и проверьте, что чат открывается.
Если вы уже вошли в Claude и видите чат, этот шаг готов.

Шаг 2

Оформите подписку

обязательна для нормальной работы на тренинге. Claude Pro стоит около $20 в месяц. Цена может отличаться по региону и налогам.

  1. Откройте официальный сайт подписки: claude.ai/upgrade.
  2. Выберите тариф Claude Pro.
  3. Проверьте, что оплата прошла успешно.
  4. Обновите страницу сервиса после оплаты.
  5. Убедитесь, что платный план отображается в аккаунте.
- это инструмент Claude для работы с кодом.

Шаг 3

Создайте аккаунт GitHub

нужен, чтобы хранить код сайта и потом опубликовать его через .

  1. Откройте официальный сайт: github.com.
  2. Нажмите регистрацию или вход.
  3. Выберите вход через , если такая кнопка доступна.
  4. Если GitHub попросит пройти , проходите ее внимательно и спокойно: лучше сделать сразу с первого раза, потому что после ошибок проверка может стать сложнее.
  5. Подтвердите email через письмо от GitHub.

Шаг 4

Создайте GitHub token

нужен, чтобы инструмент для кода смог работать с вашим : создать , отправить код и подготовить публикацию.

  1. Откройте страницу создания token: github.com/settings/tokens/new.
  2. В поле Note / Примечание напишите любое понятное название для token.
  3. В поле Expiration / Срок действия выберите No expiration.
  4. Ниже поставьте галочку рядом с repo / репозиторий.
  5. Пролистайте вниз и нажмите зеленую кнопку Generate token / Сгенерировать токен.
После того как создали , сразу сохраните его в надежном месте. GitHub покажет его только один раз.

Шаг 5

Настройте Claude Code

Теперь откройте инструмент для работы с кодом. Он понадобится на тренинге, чтобы создавать файлы проекта и отправлять код в GitHub.

  1. Откройте .
  2. Войдите в тот же аккаунт, где оформлена подписка.
  3. Откройте или выберите рабочую папку проекта.
  4. Проверьте, что инструмент видит файлы и готов принимать задачу.
  5. Если появится запрос на доступ к , разрешите его.
Достаточно убедиться, что Claude Code открывается и готов работать с файлами проекта.
