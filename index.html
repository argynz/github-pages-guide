Шаг 4

Установите

помогает приложению сохранять изменения и отправлять проект в GitHub.

  1. Скачайте установщик: Git для Windows.
  2. Откройте скачанный файл Git-2.55.0-64-bit.exe.
  3. Во всех окнах оставляйте настройки по умолчанию и нажимайте Next / Далее.
  4. В конце нажмите Install, затем Finish.
  1. Откройте : нажмите Cmd + Space, напишите Terminal и нажмите Enter.
  2. Вставьте установки :
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. Дождитесь завершения установки. Если система попросит пароль от Mac, вводите его вслепую и нажмите Enter.
  2. Затем вставьте вторую команду:
brew install git
После установки перезапустите приложение, если оно было открыто.

Шаг 7

Создайте GitHub token

позволит приложению создать и отправить туда файлы.

  1. Откройте страницу: github.com/settings/tokens/new.
  2. В Note напишите любое понятное название.
  3. В Expiration выберите No expiration.
  4. Поставьте галочку рядом с repo.
  5. Внизу нажмите зелёную кнопку Generate token.
  6. Сразу скопируйте token и сохраните его: GitHub покажет его только один раз.
Важно

Не показывайте token другим

Это отдельный ключ от GitHub. Храните его как пароль.
