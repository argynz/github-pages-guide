# Registration and GitHub Pages setup guide

Статическая страница-инструкция перед тренингом: главная страница, общий план
подготовки и интерактивный пошаговый гайд по Claude/ChatGPT, подписке, GitHub,
GitHub token и настройке Claude Code или Codex.

## Что внутри

- `index.html` - пошаговая инструкция.
- `styles.css` - оформление.
- `script.js` - переключение шагов и плавная навигация.
- `assets/workflow-hero.png` - hero-баннер.

## Как опубликовать на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое этой папки в репозиторий.
3. Откройте `Settings` -> `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/root`.
6. Нажмите `Save` и дождитесь публикации.

После deploy GitHub покажет публичную ссылку.
